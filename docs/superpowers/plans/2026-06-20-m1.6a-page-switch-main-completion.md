# M1.6a — Page-switch infrastructure + Main page completion — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add multi-page tab navigation to the editor and complete the Main sound page (conditional filter-type icons + verified bidirectional sync), as the first sub-task of M1.6.

**Architecture:** A `PageContainer` owns one `juce::Component` per `PageId` and shows exactly one at a time via `setVisible` (never rebuild/re-layout — preserves the M1.5 repaint fix). A `PageButtonBar` renders the skin's existing page buttons as a radio group that drives the container. The Main page is a real `PageComponent`; other pages are background-only `PageComponent` placeholders. A new `ConditionalImage` widget renders the `F1Type` filter-type icons via the skin's `image`+`condition` nodes.

**Tech Stack:** C++17, JUCE 8.0.13, CMake + CTest, the project's header-only `TestRunner.h` (`EXPECT` / `EXPECT_EQ`). New GUI unit tests build via `juce_add_console_app` linking `juce_gui_basics`, with a `juce::ScopedJuceInitialiser_GUI` in `main`.

## Global Constraints

- **No new third-party dependencies.** JUCE 8.0.13 + existing submodules only.
- **Switching is `setVisible` only** — never rebuild or re-layout a page on switch (NFR: no repaint storm; M1.5 regression risk).
- **`ConditionalImage` is a pure indicator** — no input, no undo entry, no SNDP/MIDI send (NFR-4: only user edits push undo).
- **Skin coordinates are scaled by 0.5** for display (`kSkinScale = 0.5f`), matching `PageComponent::skinToDisplay`.
- **All new source files** live under `source/mw2xtEditor/` and use `namespace mw2xt`, ending with `JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR`.
- **Every new test** registers via `add_test` in `tests/CMakeLists.txt` and must run green under `ctest`.
- **Deferred (do NOT implement here):** `wtMinus`/`wtPlus` Wave inc/dec buttons (→ M1.6d); graphical ADSR / wave-env / LFO icons / waveform preview (→ M1.6b–d); NAME & GLOBAL navigation (→ M1.6g/h); real content for Mod/Arp/Multi/Wave/PatchManager pages.

#include "ConditionalImage.h"
#include <algorithm>

namespace mw2xt {

ConditionalImage::ConditionalImage(juce::Image img, int idx, std::vector<int> vals)
    : image(std::move(img)), sdataIndex(idx), enableValues(std::move(vals))
{
    setInterceptsMouseClicks(false, false);
    setVisible(false);
}

void ConditionalImage::refreshFromValue(uint8_t rawValue) {
    const bool on = std::find(enableValues.begin(), enableValues.end(),
                              static_cast<int>(rawValue)) != enableValues.end();
    setVisible(on);
}

void ConditionalImage::paint(juce::Graphics& g) {
    if (image.isValid())
        g.drawImage(image, getLocalBounds().toFloat());
}

std::vector<int> parseEnableValues(const juce::String& s) {
    std::vector<int> out;
    juce::StringArray toks;
    toks.addTokens(s, " ", "");
    for (const auto& t : toks) {
        const auto trimmed = t.trim();
        if (trimmed.isNotEmpty())
            out.push_back(trimmed.getIntValue());
    }
    return out;
}

} // namespace mw2xt
```

#include "PageContainer.h"

namespace mw2xt {

void PageContainer::addPage(PageId id, std::unique_ptr<juce::Component> page) {
    if (page == nullptr)
        return;
    page->setVisible(false);
    page->setBounds(getLocalBounds());   // resized() re-applies once bounds are set
    addChildComponent(*page);            // adds as a hidden child
    slots.push_back({ id, std::move(page) });
}

void PageContainer::showPage(PageId id) {
    current = id;
    for (auto& s : slots)
        s.page->setVisible(s.id == id);
}

bool PageContainer::hasPage(PageId id) const noexcept {
    for (const auto& s : slots)
        if (s.id == id)
            return true;
    return false;
}

juce::Component* PageContainer::getPageForTest(PageId id) const noexcept {
    for (const auto& s : slots)
        if (s.id == id)
            return s.page.get();
    return nullptr;
}

void PageContainer::resized() {
    for (auto& s : slots)
        s.page->setBounds(getLocalBounds());
}

} // namespace mw2xt
```

PageButtonBar::PageButtonBar(juce::Image sheet, std::vector<ButtonDef> defs) {
    setInterceptsMouseClicks(false, true);   // pass clicks through to child buttons

    selectPage(PageId::Main);
}

void PageButtonBar::handleClick(PageId id) {
    selectPage(id);
    if (onPageSelected)
        onPageSelected(id);
}

void PageButtonBar::selectPage(PageId id) {
    selected = id;
    for (auto& e : entries)
        e.btn->setToggleState(e.id == id, false);   // false = no callback
}

bool PageButtonBar::getToggleState(PageId id) const noexcept {
    for (const auto& e : entries)
        if (e.id == id)
            return e.btn->getToggleState();
    return false;
}
